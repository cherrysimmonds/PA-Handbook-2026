# PA Handbook Assistant 🤖

An interactive Q&A chatbot for the **Independent Lives Personal Assistant Handbook** (V.1, February 2026).

Built as a standalone HTML web app — no server required. Works directly in any modern browser and can be hosted on GitHub Pages for free.

## Features

- 💬 **Natural language Q&A** — ask questions in plain English
- 🗂️ **Browse by topic** — sidebar with all handbook sections
- 🔍 **Smart keyword search** — finds the most relevant section(s)
- 📱 **Mobile responsive** — works on phones and tablets
- 🚀 **No dependencies** — single HTML file, no server or API needed

## Topics Covered

- What is a Personal Assistant?
- Code of Professional Conduct
- Working Relationship
- Starting Employment & Probationary Period
- Safeguarding & DBS Checks
- Absence, Sickness & Holiday Entitlement
- Maternity / Paternity / Parental Leave
- Data Protection
- Confidentiality & Bribery
- Social Networking Policy
- Health, Safety, Welfare & Hygiene
- Alcohol, Drugs & Smoking Policy
- Capability & Termination of Contract
- Disciplinary & Grievance Procedure
- Training Opportunities
- Useful Contacts

## Usage

### Option 1: Open Locally
Simply open `index.html` in any web browser.

### Option 2: Deploy to GitHub Pages
1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Select **Branch: main** and **Folder: / (root)**
4. Click **Save** — your app will be live at `https://<username>.github.io/<repo-name>/`

## Technology

- Pure HTML, CSS, and vanilla JavaScript
- No external libraries or frameworks
- No API keys or backend required
- Keyword-based search engine built in
